# Zesty-Food Delivery App

A full-stack MERN (MongoDB, Express, React, Node.js) food delivery application that allows users to browse, order, and manage food items online. This project is built as a portfolio project to showcase modern web development skills.

---

## Features

- User Authentication: Sign up, login, and manage user profiles.  
- Food Catalog: Browse various food items with details, images, and ratings.  
- User Ratings: Users can rate food items and leave feedback.  
- Cart System: Add items to the cart, update quantities, and view total price.  
- Order Management: Place orders and track order history.  
- Promotions: Apply promo codes to get discounts.  
- Payment Integration: Stripe test mode integration for secure payments.  
- Admin Panel: Manage food items, orders, promotions, and view customer ratings.

---

## Tech Stack

- Frontend: React, React Router, Axios, Vite, React Toastify, CSS  
- Backend: Node.js, Express.js, MongoDB Atlas  
- Authentication: JWT (JSON Web Tokens)  
- Payment: Stripe (Test mode)  
- Other Tools: GitHub, Postman (for API testing)
